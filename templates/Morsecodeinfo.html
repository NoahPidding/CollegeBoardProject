Morse code is a method used in telecommunication to encode text characters as standardized sequences of two different signal durations, called dots and dashes or dits and dahs. Morse code is named after Samuel Morse, an inventor of the telegraph.

It is not used as much today as it was in the 19th and 20th centuries. Teletypewriters were invented in the early 20th century with their own codes and gradually replaced Morse code. Other types of technology that are easier to use for communication became even more common. However, some people still use Morse code to communicate on amateur radio. There are different types of Morse code for some different countries. There are three different symbols in Morse code; there's a short one, usually called 'dit', a long one, called 'dah', and the pause. A dah is three times as long as a dit, and the pause has the same length as the dit. It's mainly used for ships to communicate.

To understand how to decrypt and encrypt morse code I needed to learn how to properly write sentences in morse code. This is done with dots and dashes in a specific order which can be translated. To code this, I first created an Encrypt function that captures an input string of letters, then parsed thru the string of alphanumeric characters one-by-one, and built up a new string of the morse code equivalent (which was stored in a pre-built array) of each character. I then printed out that morse string equivalent string. Additionally, I created an almost identical Decrypt function, with the arrays simply reversed. One change I did have to make was to create a variable for space so I had made a slash be that variable to separate words in morse code.
